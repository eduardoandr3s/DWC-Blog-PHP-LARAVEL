# 📝 Blog System - DWC (Laravel & PHP)

Este es un sistema de gestión de contenidos (Blog) desarrollado como parte del módulo de **Desarrollo Web en Entorno Servidor**. El objetivo del proyecto es implementar un CRUD completo utilizando el framework **Laravel**, siguiendo el patrón de diseño MVC y buenas prácticas.

## 🚀 Características
- **Gestión de Posts:** CRUD completo de artículos.
- **Sistema de Plantillas:** Uso de **Blade** para una interfaz dinámica.
- **Base de Datos:** Migraciones y modelos definidos mediante **Eloquent ORM**.
- **Frontend Assets:** Integración con **Vite**.

## 🛠️ Tecnologías utilizadas
- **Backend:** PHP 8.x (Laravel 10+)
- **Frontend:** Blade Templates & CSS3
- **Base de Datos:** MySQL
- **Herramientas:** Composer, Artisan, Vite

## 📦 Instalación
1. `git clone https://github.com/eduardoandr3s/DWC-Blog-PHP-LARAVEL.git`
2. `composer install`
3. `npm install && npm run dev`
4. Configurar `.env` y `php artisan key:generate`
5. `php artisan migrate`
6. `php artisan serve`
